# New_Vacancy Backend — Supabase Setup Guide

## Stack
- **Database**: Supabase PostgreSQL
- **Auth**: Supabase Auth (Email + Google OAuth)
- **API**: Supabase Auto-generated REST + Edge Functions
- **Storage**: Supabase Storage (for banners/images)

---

## 1. Create Supabase Project

1. Go to [https://supabase.com](https://supabase.com) → New Project
2. Name: `new-vacancy`
3. Region: **South Asia (Mumbai)** — closest to India users
4. Save your `SUPABASE_URL` and `SUPABASE_ANON_KEY`

---

## 2. Run Migrations

In Supabase Dashboard → **SQL Editor**:

```sql
-- Run in order:
-- 1. migrations/001_initial_schema.sql
-- 2. migrations/002_seed_data.sql
```

Or use Supabase CLI:
```bash
npm install -g supabase
supabase login
supabase link --project-ref YOUR_PROJECT_REF
supabase db push
```

---

## 3. Configure Auth

In Supabase Dashboard → **Authentication → Providers**:

- Enable **Email** (with email confirmation)
- Enable **Google** OAuth:
  - Add Client ID + Secret from Google Cloud Console
  - Redirect URL: `https://YOUR_DOMAIN/auth/callback`

---

## 4. Set Admin User

After your first signup, run in SQL Editor:
```sql
UPDATE public.profiles 
SET role = 'admin' 
WHERE email = 'your-admin@email.com';
```

---

## 5. Environment Variables (for frontend)

```env
VITE_SUPABASE_URL=https://xxxxxxxxxxxx.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key-here
```

---

## 6. Storage Buckets (for images)

In Supabase Dashboard → **Storage**:
```
Create buckets:
- job-images (public)
- news-images (public)
- affiliate-banners (public)
```

---

## API Endpoints (Auto-generated by Supabase)

| Resource | Endpoint |
|---|---|
| Jobs | `GET /rest/v1/jobs?category=eq.govt&order=posted_at.desc` |
| News | `GET /rest/v1/news?category=eq.tech&order=published_at.desc` |
| Affiliates | `GET /rest/v1/affiliates?is_active=eq.true&order=position` |
| Saved Jobs | `GET /rest/v1/saved_jobs?user_id=eq.USER_ID` |

All endpoints require `apikey` header with your anon key.
Authenticated endpoints also require `Authorization: Bearer USER_JWT` header.
